# bounce

A multiplayer browser game built with SvelteKit, WebGPU, and WebSockets.

## Tech Stack

- **Frontend** — SvelteKit + Svelte 5 + TypeScript
- **Rendering** — WebGPU + WGSL
- **Multiplayer** — WebSockets (Bun)
- **Runtime** — Bun

## Requirements

- [Bun](https://bun.sh) v1.0+
- A WebGPU-compatible browser (Chrome 113+, Edge 113+)

## Setup

```bash
# install dependencies
bun install

# start dev server
bun dev
```
